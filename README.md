# OpenMRS Drools Module (CDS Engine)

This module provides a **Clinical Decision Support (CDS) engine for OpenMRS** powered by [Drools](https://www.drools.org/).  
It allows implementers and developers to define and evaluate rules against OpenMRS data, supporting use cases such as:

- Patient flags
- Alerts & notifications
- Task list management
- Rule-driven forms and recommendations
- Etc

📖 For detailed documentation, see the [Wiki](https://openmrs.atlassian.net/wiki/spaces/projects/pages/603750673/OpenMRS+CDS+Engine+powered+by+Drools).

---

## Requirements

- **Java**: JDK 11+ (Drools 9.44.0.Final minimum requirement)  
- **OpenMRS**: Platform 2.6.x+
